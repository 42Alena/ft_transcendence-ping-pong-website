Setup instructions for teammates:

0. Node.js

Node.js is a JavaScript runtime used to run backend code (outside the browser).
It is required to run Fastify server, real-time features (Socket.IO), and backend logic inside Docker.

1. Installing packages inside /backend:

cd backend  
npm init -y                 # Create package.json  
npm install fastify socket.io  
npm install -D typescript ts-node @types/node @types/socket.io

This installs the main server (Fastify), real-time support (Socket.IO), TypeScript compiler, and type definitions.

2. Create backend source folder:

- Create folder: `backend/src`
- Create file: `src/main.ts` (entry point)

---

Local development:

1. Install latest Node.js (version 24 or higher)
2. Navigate to the backend folder:
   cd backend  
3. Install dependencies:
   npm install  
4. Run server:
   npm run start  
5. Check for TypeScript type warnings only:
   npm run tsc  

---

With Docker:

1. Build Docker images:
   docker compose build  
   # or only backend:
   docker compose build backend  

2. Run backend service:
   docker compose up backend  

3. Run in background:
   docker compose up -d backend  

The backend will be available at http://localhost:3000/ping or /hola
