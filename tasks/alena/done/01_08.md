### 1. Initialized frontend project in `/frontend`

```sh
cd frontend
npm init -y
```
---
### 2. Installed required dependencies
```sh
npm install react react-dom socket.io-client
npm install -D typescript vite tailwindcss postcss autoprefixer
npm install -D @types/react @types/react-dom
```
---
### 3. Tailwind Setup
Since npx tailwindcss init -p failed, created config files manually:

    tailwind.config.js
    postcss.config.js

Also created Tailwind base file:

touch src/index.css

With content:

@tailwind base;
@tailwind components;
@tailwind utilities;

### 4. Vite Setup
Created:

    index.html in /frontend

    Installed plugin:

npm install -D @vitejs/plugin-react

Created vite.config.ts:

import { defineConfig } from 'vite'

export default defineConfig({
  plugins: [react()],
})
